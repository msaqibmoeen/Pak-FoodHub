# 🍽️ Pak Food Hub

Pak Food Hub is a web-based platform that showcases popular Pakistani foods from different regions. The goal of this project is to promote Pakistani cuisine and provide users with information about traditional dishes in one place.

---

## 🚀 Features

- 🍛 List of famous Pakistani foods
- 🗺️ Regional food categorization
- 📖 Short description of each dish
- 🔍 Easy navigation and user-friendly UI
- 📱 Responsive design (mobile + desktop)

---

## 🧑‍💻 Tech Stack

- HTML5  
- CSS3  
- JavaScript

---

## 🍲 Popular Foods Included

- Biryani  
- Nihari  
- Karahi  
- Haleem  
- Seekh Kabab  
- Pulao  
- Samosa & Snacks  
- Halwa Puri  

---

## 🎯 Purpose

This project is created to:
- Highlight Pakistani food culture
- Practice frontend development skills
- Build a clean and simple food showcase website

---

## 📌 How to Run

1. Clone the repository  
2. Open `index.html` in your browser  
3. Explore Pakistani food items
